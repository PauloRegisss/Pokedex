🐾 Minha Jornada Criando uma Pokédex 🐾

Bem-vindo(a) à minha Pokédex interativa! Aqui compartilho o caminho que percorri para transformar uma ideia inspirada no universo Pokémon em um projeto funcional utilizando HTML, CSS e JavaScript.

💻 Tecnologias Utilizadas

HTML: Estruturei a Pokédex com seções organizadas, cards para cada Pokémon e campos de busca dinâmicos.

CSS: Dei vida ao projeto com cores vibrantes, sombras sutis, efeitos de hover e responsividade para uma experiência envolvente.

JavaScript: O coração da Pokédex! Usei JS para buscar dados da API Pokémon, manipular o DOM, criar filtros e permitir interações em tempo real.

🚀 Passo a Passo da Jornada

Planejamento e Layout
Comecei desenhando no papel a estrutura principal: barra de pesquisa, área de exibição dos Pokémon e cards individuais. Defini cores, fontes e estilos que lembrassem o universo Pokémon.

Estrutura HTML
Criei os elementos básicos da Pokédex: cabeçalho, input de busca, área de cards e footer. Cada Pokémon seria representado por um card dinâmico, com nome, tipo e imagem.

Estilização com CSS
Brinquei com cores e gradientes inspirados nos Pokémons, usei flexbox e grid para organizar os cards e apliquei animações sutis para tornar a experiência mais interativa.

Interatividade com JavaScript
Aqui entrou a magia:

Conexão com a PokéAPI
 para buscar dados atualizados.

Renderização dinâmica dos cards com as informações recebidas.

Filtros e barra de pesquisa para encontrar rapidamente qualquer Pokémon.

Tratamento de erros, garantindo que o usuário sempre recebesse uma experiência suave.

Testes e Ajustes Finais
Ajustei responsividade para diferentes telas, refinei cores, organizei o código e garanti que todos os Pokémon fossem carregados corretamente.

🎨 O Resultado

O projeto final é uma Pokédex funcional, intuitiva e visualmente agradável, pronta para explorar e se divertir com qualquer fã de Pokémon ou entusiasta de front-end.
